# Bus Stop Sun Shade Simulator

This is a simple web-based simulator to visualize the shadows cast by a bus stop shelter throughout the day.

## How to run

Simply open the `index.html` file in a modern web browser that supports WebGL.

## Features

- 3D model of a bus stop shelter.
- Realistic sun position calculation based on latitude and longitude (default is Zagreb, Croatia).
- Animated sun path from sunrise to sunset.
- Real-time shadow casting.
- Orbit controls to view the scene from different angles (drag with mouse, scroll to zoom).

## Technologies used

- HTML5
- CSS3
- JavaScript
- [three.js](https://threejs.org/) for 3D graphics.
- [suncalc.js](https://github.com/mourner/suncalc) for sun position calculations.
